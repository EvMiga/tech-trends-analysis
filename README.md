# tech-trends-analysis
Analysis of current and emerging technologies and trends in the software development industry.

**tech-trends-analysis** is a comprehensive research and analytical project dedicated to exploring current and future trends in the software development industry.

This repository contains not only source code, but also data analysis, visualizations, research findings, and a presentation summarizing key insights.

The project examines how technologies evolve, how they are distributed globally, and how education influences participation in the tech industry.

___________________________________________

The main objectives of this project are:

- Analyze current and emerging programming languages
- Identify the most relevant and modern database technologies
- Examine regional distribution of developers worldwide
- Explore the relationship between education level and employment in the technology sector
- Identify skills and technologies that will remain in demand in the future

___________________________________________________
